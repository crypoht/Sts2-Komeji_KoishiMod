⚙️ 核心机制 (Core Mechanics)
👁️ 无意识 (Unconscious)
全新的卡牌 Tag。

本身没有任何特殊效果，仅作为标记。带有此标记的卡牌可以在特定的效果下被自动打出。

💠 弹幕 (Danmaku)
专属的衍生卡体系，共有 9 种 不同效果的弹幕：

爱心弹幕：造成伤害，并给予敌人“虚弱”。

星星弹幕：造成较高的基础伤害。

阴阳玉弹幕：造成巨量伤害，不消耗（耗能为 3 费）。

米粒弹幕：可以复制自身。

小玉弹幕：造成多段连击伤害。

大玉弹幕：给予敌人“易伤”。

箭形弹幕：对所有敌人造成群体伤害。

方形弹幕：给予敌人特殊的“追踪”状态。

蔷薇弹幕：稀有弹幕，不会被一般的“生成弹幕”卡牌随机生成。

🎯 追踪 (Tracking)： 专属异常状态。拥有此状态的敌人，受到来自“弹幕”类卡牌的伤害增加 50%。

🌸 盛开与闭合 (Bloom & Close Stances)
类似于观者的“暴怒”与“平静”，恋恋拥有自己的专属姿态循环：

盛开 (Bloom)：

效果： 打出攻击牌时，额外对随机敌人造成等同于该牌伤害的伤害。但获得的格挡值减少 50%。

入场奖励： 进入该状态时，获得 1 点能量。

闭合 (Close)：

效果： 获得的格挡值提升 20%。

入场奖励： 进入该状态时，抽 1 张牌。

(开发者碎碎念：目前这套姿态的实战数值有点太高了，估计以后会削弱。做完这个终于知道为什么观者那么强了！)
