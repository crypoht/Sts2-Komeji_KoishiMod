# Sts2-Komeji_KoishiMod
"Adds a new playable character, Komeiji Koishi, to Slay the Spire 2."
添加了新角色古明地恋,来自东方project

目前多人测试没有多测试几次,估计bug很多,有不过可以通过**179669710@qq.com**联系我,可以的话报日志也发过来(win+r后输入%appdata%\SlayTheSpire2\logs)

机制:
*无意识*
新卡牌tag,本身没有特殊效果,作为标记,可以被在特定效果下自动打出

*弹幕*
衍生卡,有九种,分别为:
爱心弹幕:造成伤害并给予虚弱
星星弹幕:造成较大伤害
阴阳玉弹幕:造成大量伤害,不消耗,3费
米粒弹幕:可以复制自己
小玉弹幕:造成多段伤害
大玉弹幕:给与易伤
箭形弹幕:对所有敌人造成伤害
方形弹幕:给予追踪
蔷薇弹幕:不会被一般弹幕生成卡

追踪:弹幕类卡造成的伤害增加50%

*$\color{red}{\text{盛开}}$与$\color{blue}{\text{闭合}}$*:
盛开:打出攻击牌时，额外对随机敌人造成等同于该牌伤害的伤害。获得的格挡减少 50%。进入该状态时,获得一点能量
闭合:"获得的格挡提升 20%。进入该状态时抽取一张牌"

类似于观者的暴怒与平静,但实际打得时候数值太高了,估计以后会改(终于知道为什么观者这么强)

目前多人手没画(恋恋隐身很正常吧),先古遗物及其卡没有弄,所以进入二层可能卡死(明天就修),多人卡为无
